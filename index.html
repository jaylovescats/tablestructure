html tables are used to display data in a structured format of rows and columns. they help organise information clearly, making it easier to read and compare. tables are useful for things like schedules, data summaries, or comparisons, and can be styled with css for a more polished look.

table elements

tag name description
<table> table defines a table
<th> table head defines a header cell in a table
<tr> table row defines a row in a table
<td> table data defines a cell in a table
<caption> caption defines a table caption
<colgroup> column group specifies a group of one or more columns in a table for formatting
<col> column specifies column properties for each column within a <colgroup> element
<thead> table header groups the header content in a table
<tbody> table body groups the body content in a table
<tfoot> table footer groups the footer content in a table
